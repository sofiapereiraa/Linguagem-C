# 📚 Projetos em Linguagem C

Este repositório contém pequenos projetos desenvolvidos em **linguagem C**.

---

## 🧠 Projetos incluídos

### 🧮 Calculadora simples
Programa com menu interativo para operações básicas entre dois números.

**Funcionalidades:**
- Soma
- Subtração
- Multiplicação
- Divisão (com verificação de divisão por zero)
- Menu com escolha de operação

---

### 🏦 Sistema bancário simples
Simula operações básicas de uma conta bancária.

**Funcionalidades:**
- Consulta de saldo
- Saque com validação de saldo
- Depósito

---

### 📊 Média escolar
Calcula a média de trabalhos e provas e define a situação do aluno.

**Regras:**
- Média dos trabalhos
- Média ponderada das provas
- Média final

---

### 👤 Verificador de idade
Classifica a idade informada em categorias.

---

### ⏳ Temporizador 

Temporizador desenvolvido para o usuário informa minutos e segundos, e o programa realiza a contagem até 00:00, emitindo um alerta sonoro ao finalizar.

---

# 📦 Sistema de Cadastro de Produtos

Sistema de cadastro de produtos simples.

## 🚀 Funcionalidades

- ➕ Adicionar produto (nome, preço e quantidade)
- 📋 Listar produtos cadastrados
- 🚪 Sair do programa
- 📋 Menu interativo para navegação
