# 📚 Biblioteca 
Projeto desenvolvido em linguagem C, simula um sistema de biblioteca simples. O programa permite cadastrar, listar, buscar, emprestar e devolver livros através de um menu interativo no terminal.

---

## Funcionalidades
- Cadastro de livros (Nome do livro,Autor, Editora e Ano de publicação)
- Listagem de livros cadastrados
- Busca de livros por título
- Empréstimo de livros
- Devolução de livros
- Geração automática de ID para cada livro
- Menu interativo para navegação entre as opções
- Encerramento do programa

---

## 🛠️ Tecnologias Utilizadas
- Linguagem C
- Biblioteca stdio.h
- Biblioteca string.h

---

## ▶️ Compilação
No terminal:
gcc sistema.c -o sistema

▶️ Execução
./sistema
