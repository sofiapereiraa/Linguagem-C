# 📝 Diário de Lembretes em C

Projeto desenvolvido em linguagem C para praticar manipulação de arquivos texto, entrada e saída de dados e modularização utilizando funções.

O sistema funciona como um pequeno diário de lembretes, permitindo que o usuário registre anotações e as consulte posteriormente por meio de um arquivo `.txt`.

---

## 🚀 Funcionalidades

- ✅ Inserir um ou mais lembretes
- ✅ Armazenar os lembretes em um arquivo de texto (`diario.txt`)
- ✅ Ler todos os lembretes salvos
- ✅ Menu interativo

---

## 💻 Tecnologias e conceitos utilizados

- Linguagem C
- Manipulação de arquivos texto
- Entrada e saída de dados
- Funções
- Estruturas de repetição e decisão
- VS Code

---

## ▶️ Como executar

Compile o arquivo utilizando um compilador C, como o GCC.

```bash
gcc nome_do_arquivo.c -o nome_do_programa
./nome_do_programa
```
