# 📒 Agenda de Contatos

Projeto desenvolvido em linguagem C para gerenciamento de contatos e prática de conceitos mais avançados da linguagem. O sistema permite cadastrar, consultar, editar e remover contatos através de um menu interativo no terminal.
---

## 🚀 Funcionalidades

- ✅ Adicionar contatos
- ✅ Listar contatos cadastrados
- ✅ Buscar contatos pelo nome
- ✅ Editar informações de contatos
- ✅ Excluir contatos
- ✅ Validar nome, email e telefone
- ✅ Salvar contatos em arquivo `.txt`
- ✅ Visualizar contatos salvos em arquivo

---

## 💻 Tecnologias e conceitos utilizados

- Linguagem C
- Structs
- Manipulação de strings
- Alocação dinâmica de memória (`malloc` e `realloc`)
- Liberação de memória (`free`)
- Manipulação de arquivos (`FILE`)
- Funções
- Validação de dados
- Menu interativo
- VS Code

---

## ▶️ Como executar

Compile o arquivo utilizando um compilador C, como o GCC.

```bash
gcc nome_do_arquivo.c -o nome_do_programa
./nome_do_programa
```
